# examples/ — worked outputs

Real, end-to-end demonstrations of the brand system in use: not invented skill chains, but the
actual path from exploration to governed final assets, spanning Assignments 2 through 6.

- `tube-system-build/` — how the tube system went from moodboard to final governed asset. Shows
  the intended workflow: **explore → test → break through → lock → generate → govern.**
- `packaging-build/` — how the outer box was built once the tube system was locked. A distinct
  failure mode (natural language fell short) and fix (find + strip a reference, then reuse it).
