# Typography

## The system

| Typeface | Use |
| --- | --- |
| Cedarville Cursive | Wordmark / logo only. Never used as headline type. |
| DM Sans (Regular / Light / Extralight) | Headlines, subheads, body, captions — see scale below |
| DM Mono (Light) | Technical/label text only — tube labels, phase indicators, product codes |

## Full type scale

| Level | Weight | Use | Example |
| --- | --- | --- | --- |
| Wordmark | Cedarville Cursive | Logo only, never headline type | Celeste |
| Headline | DM Sans Regular | Hero sections, campaign headlines, full-bleed ads | "your skin is not the same throughout your cycle." |
| Subhead | DM Sans Light | Section titles, ad subheads, email headers | "built to cycle with you." |
| Body | DM Sans Regular | Website body, product descriptions, email copy | Full paragraph copy |
| Caption | DM Sans Extralight | Website body, product descriptions, email copy | "phase 03 of 04 — ovulatory. skin is at peak glow. opal supports balance." |
| Technical / Label | DM Mono Light | Tube label, phase indicators, product codes only | "03 OPAL / OVULATORY PHASE" |

## Rules

- Cedarville Cursive is reserved for the `Celeste` wordmark — do not use it for any other text.
- DM Mono Light signals "technical" — product codes, phase labels, spec sheets.
- DM Sans carries everything user-facing outside the wordmark itself, with weight (Regular/Light/
  Extralight) signaling hierarchy from headline down to caption.
