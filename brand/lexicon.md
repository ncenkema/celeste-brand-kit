# Lexicon

## Words we use

- cycle, phase, protocol, built for, moves with you
- Mend, Float, Opal, Lull (product names)
- menstrual, follicular, ovulatory, luteal (phase names)

## What copy avoids (words and phrasing)

- Spiritual wellness language
- Generic stress/self-care routine messaging
- Hero ingredient stories
- Static skin type thinking
- Em dashes (use a comma, a period, or rewrite the sentence)
- Never write like: a spiritual wellness brand, an empowerment brand, a clinical dermatology brand,
  or generic clean beauty

## Naming conventions (asset labels)

Non-dictionary, all-caps, product-code-prefixed codes — prevents AI semantic drift during
generation. Established pattern from the visual production system:

- Product prefixes: `MND` (Mend), `FLT` (Float), `OPL` (Opal), `LUL` (Lull), `ALL` (all four),
  `BLBD` (billboard)
- Version suffixes: `2`, `V1` for revised states
- Examples: `MNDCUT`, `OPLREF`, `OPLFRONT`, `FLTANGLE2`, `BLBD2`

## Brand name casing

**Celeste** — capitalized, always, including in running copy.
