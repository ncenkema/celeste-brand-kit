# assets/ — the media library

Unlike Evan's repo (which points to an external Dropbox library), Celeste's final images are
stored directly in this folder, since everything ships as one zip into the vibe-coding tool. See
[`manifest.json`](manifest.json) for the full indexed list.

## Folders

| Folder | What's in it |
| --- | --- |
| `tubes/` | Tube-alone shots, tube-on-colored-background shots, the 4-tube stack |
| `squeeze-drip/` | Tube-with-product-squeeze shots, the all-4 squeeze composite |
| `swatches/` | Individual formula swatches (4) + the all-4 swatch composite |
| `packaging/` | Box alone, box 3/4 angle, box horizontal, box+tube pair — all 4 products |
| `people/` | Lifestyle/people shots using Celeste product |
| `billboard/` | Finished billboard composites (2 versions) |
| `booth/` | Finished product display booth composite |

## Naming convention

All-caps, product-code-prefixed (`MND`, `FLT`, `OPL`, `LUL`, `ALL`, `BLBD`), non-dictionary codes to
prevent semantic drift during AI generation — same convention used throughout `brand/`.
